# Changelog

## 0.1.1

- マーケットプレイス表示用アイコンを改善（縦横比を維持したレターボックス）
- 結果ツリーで一致ファイルを右クリックし、絶対パス・ワークスペース相対パスのコピー／OS のファイルマネージャで表示ができるようにした
- 「設定を開く」が正しい拡張機能 ID（`@ext:`）を参照するよう修正
- 進行通知・Output チャンネル名を「Grep Multi Encode」に統一し、検索エラー時は Output を自動表示
- コマンドパレットから検索サイドバーにフォーカスできるコマンドを追加
- コマンドパレット検索の入力プロンプトを UI 言語に合わせて表示

## 0.1.0

- Grep Multi Encode の初回リリース
- 複数エンコード（UTF-8 / Shift_JIS / EUC-JP ほか）を横断するワークスペース検索に対応
- 一時除外入力・`.gitignore` 反映・結果ツリー表示・行ジャンプなど、実運用向けの検索体験を提供
- 設定画面のローカライズ（日英）と、主要 UI の整理・軽量化を実施
